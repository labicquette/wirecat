# **Wirecat** 

### *Wirecat is the result of the merging between **Wireshark** and **cat** from unix*
## **How to install ?**
### Requirements :

```
pip install tkinter 
```
#### And python >= 3
### Just clone the repo... And done, everything works out of the box.


### **How to execute ?** 

#### With GUI :
```
python wirecat.py 
```

#### Without GUI :
```
python wirecat.py -ng yourfile.txt
```

#### With file output :
```
python wirecat.py -o yourfile.txt
```
